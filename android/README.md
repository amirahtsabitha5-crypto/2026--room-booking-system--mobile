# Room Booking System - Android

Mobile application untuk Room Booking System yang dikembangkan menggunakan Android Native.

## Prerequisites

- Android Studio 2023.1 atau lebih baru
- Android SDK 28 (API level 9.0) atau lebih baru
- Java 11 atau lebih baru

## Setup

1. Clone repository dan buka project di Android Studio
2. Install dependencies melalui Gradle
3. Configure API endpoint di `build.gradle`
4. Run emulator atau connect physical device
5. Build dan run aplikasi

## Features

- View available rooms
- Create booking reservations
- View booking history
- Manage bookings
- Real-time booking status updates

## Architecture

- MVVM pattern
- Clean Architecture principles
- Retrofit untuk API calls
- Room Database untuk local caching

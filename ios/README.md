# Room Booking System - iOS

Mobile application untuk Room Booking System yang dikembangkan menggunakan Swift.

## Prerequisites

- Xcode 15.0 atau lebih baru
- macOS 13 atau lebih baru
- iOS 14.0 atau lebih baru (minimum deployment target)
- Swift 5.9 atau lebih baru

## Setup

1. Clone repository dan buka project di Xcode
2. Install dependencies menggunakan CocoaPods:
   ```bash
   pod install
   ```
3. Configure API endpoint di `Config.swift`
4. Run aplikasi di simulator atau physical device

## Features

- View available rooms
- Create booking reservations
- View booking history
- Manage bookings
- Real-time booking status updates

## Architecture

- MVVM pattern
- Clean Architecture principles
- Alamofire untuk API calls
- CoreData untuk local storage
